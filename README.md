# dotfiles
configs and automation to setup shop

### Prerequisites
A `Rakefile` is provided that automates the tasks of getting submodules and sym linking files. In order to use it, the system needs a version of ruby that contains rake.

Install [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh#getting-started)

### Automated
1. Clone the repo `git clone https://github.com/elauqsap/dotfiles.git`
2. Run `rake init`
3. Wait for modules to be downloaded and linked

### Manual
The following need manual intervention in the current build of these dotfiles.
